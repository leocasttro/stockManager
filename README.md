# stockManager
## Sistema de Gerenciamento de Estoque  

Este sistema permite o gerenciamento de estoque de uma empresa, oferecendo funcionalidades como:  
- Cadastro de usuários  
- Controle de estoque  
- Gerenciamento de pedidos  
- Auditorias  

### Tecnologias Utilizadas  
- Java (sem Spring Boot)  
- Maven
